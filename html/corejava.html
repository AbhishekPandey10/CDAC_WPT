Core Java, Volume I: Fundamentals by Cay S. Horstmann: This book is aimed at programmers with some experience who want to learn Java for practical application. It covers the core concepts of the language in depth, including object-oriented programming, exceptions, generics, collections, and more. Users consider it a comprehensive and in-depth resource for mastering Java fundamentals [Amazon in, Core Java : Horstmann, Cay].

Core Java for the Impatient by Cay Horstmann: This book is a fast-paced introduction targeted at programmers already familiar with another language. It focuses on the modern features of Java, like lambdas, streams, and modules, and provides a concise reference for experienced developers [Cay Horstmann, Core Java for the Impatient].
